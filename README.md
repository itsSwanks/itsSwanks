## 🚀 About Me

- 🍎 Independent developer — I build **native Apple apps** for iPhone, iPad & Apple Watch
- 📦 On the App Store: **[Receira](https://apps.apple.com/us/app/receira-for-papra/id6774353903)** — an open-source (AGPL-3.0) client for the self-hosted **Papra** document server
- 🧪 Also building **TYP — Track Your Pep**, a private, offline-first tracker
- 🌐 Everything I ship docks at **[itsswanks.github.io](https://itsswanks.github.io)**
- 💬 Ask me about Swift, SwiftUI, offline-first sync, or shipping to the App Store
